DagsHub Admin API (1.0.0)

Download OpenAPI specification:

Admin API for on-premise DagsHub sysadmin automation. All endpoints require admin privileges.

Authentication

Use HTTP Basic authentication with an admin account's username and password or access token:

Authorization: Basic base64(username:token)

Access Control

All endpoints in this spec are protected by admin middleware. Requests from non-admin accounts will receive a 403 response.

LabelStudio

Remove a repository's annotation workspace

Asynchronously removes the Label Studio workspace associated with the repository. Use this to clean up workspaces that are stuck or need to be reprovisioned.

The service query parameter selects the workspace type:

  • 0 = Label Studio (Git backend)
  • 1 = Label Studio (S3 backend)
Authorizations:
basicAuthtokenAuth
path Parameters
owner
required
string
repo
required
string
query Parameters
service
integer
Default: 0

Workspace service type: 0 = LS Git, 1 = LS S3

Responses

https://{host}/api/v1/admin/repos/{owner}/{repo}/annotations/remove-workspace
